## NOTE
> This script will update every day if I'm not busy

### FOR TERMUX USER
```bash
> pkg update && pkg upgrade
> pkg install git -y
> pkg install nodejs -y
> pkg install ffmpeg -y
> pkg install imagemagick -y
> git clone https://github.com/MuhammadRestu999/RestuBot
> cd RestuBot
> npm install
```
###### Run
```bash
> node index
```
